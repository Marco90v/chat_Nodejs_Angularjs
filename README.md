# chatNodejsAngularjs
 Chat web con Nodejs y Angularjs

 Esta es una pequeña practica realizada para aprender NodeJs y poner en practica los conocimientos previamente adquiridos de AngularJS

 La app es simple crea una sesión de usuario, verifica si el nombre de usuario fue creado previamente.
 Los mensajes se redirecciona a todos los usuarios, se detecta quien envía el mensaje para asignar estilos a los mensaje.
 cierra sesión, borra mensajes.
